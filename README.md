# LAB1_Devmobile
Application Android simple développée en Java avec Android Studio, conçue pour implémenter une activité avec des boutons et textView .

L'application présente une interface composée de :
  -Une barre de titre bleue affichant "Hello Toast"
  -Un bouton TOAST en haut
  -Une zone d'affichage jaune centrale montrant un compteur numérique
  -Un bouton COUNT en bas

# Boutons
TOAST Affiche un message de bienvenue sous forme de notification temporaire (Toast) ----------
COUNT Incrémente le compteur affiché dans la zone jaune à chaque clic
